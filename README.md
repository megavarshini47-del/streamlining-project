Project Title : Streamlining Ticket Assignment for Efficient Support Operations

Project Overview:
        This project focuses on improving support operations by automating the process of ticket assignment in ServiceNow.In many organ;izations, support tickets are assigned manually, which often leads to delays, incorrect routing, and confusion about responsibility. This project solves that problem by implementing an automated system using ServiceNow features like users, groups, roles, ACLs, and flows.

Problem Statement : 
      1.Manual ticket routing
      2.Delayed issue resolution
      3.Misassigned tickets
      4.Lack of visibility on responsibility
      
Implementation Phases:
      Phase 1: Create Users
      Phase 2: Create Groups
      Phase 3: Assign Roles
      Phase 4: Create Custom Table
      Phase 5: Automate with Flows
      
Outcome :
      1.Reduced manual work in ticket handling
      2.Faster and more accurate ticket assignment
      3.Improved visibility and accountability
      4.Enhanced overall support efficiency
      
Conclusion:
This project demonstrates how automation in ServiceNow can significantly improve support operations by reducing errors and saving time. It highlights the importance of structured user roles, organized groups, and workflow automation in modern IT service management.
